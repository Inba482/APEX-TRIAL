# 📖 Project Overview

APEX TRIAL is a modern e-commerce product catalog that enables users to browse products, search items, filter by categories, view product details, and manage a shopping cart through a clean and responsive interface.

The project is designed as a frontend capstone demonstrating professional web development practices suitable for internship and portfolio presentation.

# 🏗 Architecture

The project follows a modular frontend architecture.

- Components are reusable.
- JavaScript is separated into modules.
- Product data is loaded from JSON.
- Local Storage manages the shopping cart and wishlist.
- The structure is designed for scalability and maintainability.
